# Review Router

**Use directly:**

- **`review-bugbot`** — Code review skill: catches bugs, regressions, performance issues
- **`review-security`** — Security review skill: OWASP Top 10, auth/XSS/SQL injection checks

Both skills define the review methodology. Subagents are the workers.
